# Object JS Project

This folder contains a small JavaScript project demonstrating object-oriented programming and DOM interaction.

## Files

- `index.html` — the main page that loads the JavaScript and displays the interactive content.
- `script.js` — JavaScript code for object handling and interaction logic.
- `style.css` — styling for the page layout and appearance.
- `images/` — image assets used by the project.

## Description

This project focuses on working with JavaScript objects and connecting them to HTML elements. It is built with plain HTML, CSS, and JavaScript.

## How to use

1. Open `index.html` in your browser.
2. Or serve the folder from a local web server.
3. Edit `script.js` to update object logic and interaction behavior.

## Notes

- The project is meant to show JavaScript object usage in a practical web page.
- Use the CSS file to customize the visual style.
- Add more examples or object patterns directly in `script.js`.
